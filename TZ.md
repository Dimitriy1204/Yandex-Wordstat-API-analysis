## 1. Общие сведения

### 1.1. Наименование системы

**WordstatAgent** (рабочее имя: *Yandex Wordstat Analysis Agent*) — локальное веб-приложение для маркетологов и аналитиков, сравнивающее поисковые запросы по данным Yandex Wordstat.

### 1.2. Основание для разработки

Необходимость автоматизировать рутинный анализ ниш: сбор частотности, расчёт метрик, визуализация, прогноз, формирование отчёта в едином корпоративном формате Excel и опциональная интерпретация результатов нейросетью YandexGPT.

### 1.3. Цели создания системы

1. Сократить время подготовки сравнительного отчёта по поисковым запросам с часов до минут.
2. Обеспечить единообразное оформление отчётов по образцу `yandex_analysis_exemple.xlsx`.
3. Выявлять перспективные ниши («голубые океаны») и растущие рынки на основе объективных метрик.
4. Дать возможность работы без установки Python (режим standalone `.exe` для Windows).
5. Обеспечить опциональное использование YandexGPT без блокировки базового сценария.

### 1.4. Пользователи

| Роль | Описание |
|------|----------|
| **Аналитик / маркетолог** | Вводит запросы, запускает анализ, скачивает Excel, просматривает графики |
| **Администратор** | Один раз настраивает API-ключи, каналы отправки, прокси |

Аутентификация пользователей внутри приложения **не требуется** (локальный инструмент).

## 2. Требования к режимам работы

### 2.1. Режим разработки

- Запуск через `python run.py`.
- Веб-интерфейс на `http://127.0.0.1:8000` (порт настраивается переменной окружения).

### 2.2. Режим распространения (целевой)

- Один исполняемый файл `yandex_wordstat_agent.exe` (Windows, без консоли).
- Автоматическое открытие браузера при старте.
- Все зависимости (включая Prophet/CmdStan, шаблон Excel, HTML-интерфейс) **встроены в exe**.
- Настройки и отчёты сохраняются рядом с exe.

### 2.3. Режимы анализа

| Режим | Условие | Результат |
|-------|---------|-----------|
| **Базовый** | Ключи Wordstat, LLM выключен | Полный Excel + графики без блоков YandexGPT |
| **Расширенный** | Ключи Wordstat + LLM включён + доступ к YandexGPT | Дополнительно: расширение запросов, обзор, рекомендации, вердикты нейросети |

Переключатель LLM **по умолчанию выключен**.

## 3. Функциональные требования

### 3.1. Настройки (FR-01)

Система должна предоставлять единую веб-страницу настроек со следующими разделами:

| Раздел | Параметры | Обязательность |
|--------|-----------|----------------|
| Yandex Wordstat | API-ключ, Folder ID, endpoint | **Обязательно** для анализа |
| YandexGPT | API-ключ, Folder ID, модель, кнопка «Проверить» | Опционально |
| Telegram | Token бота, chat_id | Опционально |
| VK | access_token, peer_id | Опционально |
| Email (SMTP) | host, port, user, password, from, to, TLS | Опционально |
| Прокси | type, host, port, user, password | Опционально |
| Параметры анализа | top_n (графики), request_delay (сек.) | Опционально (есть значения по умолчанию) |

**FR-01.1.** Настройки сохраняются в зашифрованный файл `settings.enc` (AES-256, Fernet + PBKDF2).  
**FR-01.2.** Секретные поля (ключи, пароли) не отображаются в открытом виде при повторной загрузке.  
**FR-01.3.** Если ключи YandexGPT не заданы, для GPT используются ключ и Folder ID Wordstat (fallback).  
**FR-01.4.** Статус готовности Wordstat и YandexGPT отображается индикатором в интерфейсе.

---

### 3.2. Ввод данных и запуск анализа (FR-02)

**FR-02.1.** Пользователь вводит список поисковых запросов (через запятую или с новой строки).  
**FR-02.2.** Пользователь задаёт период анализа (дата начала и окончания).  
**FR-02.3.** Пользователь может включить/выключить:
- использование YandexGPT;
- отправку в Telegram / VK / Email.

**FR-02.4.** Кнопка «Запустить анализ» доступна только при настроенных ключах Wordstat.  
**FR-02.5.** Во время анализа отображается прогресс (шаг, сообщение, индикатор).  
**FR-02.6.** Анализ можно отменить; при отмене процесс останавливается без падения приложения.  
**FR-02.7.** Повторный запуск анализа блокируется, пока выполняется текущий.

---

### 3.3. Сбор данных Wordstat (FR-03)

**FR-03.1.** Источник: Yandex Search API v2, метод динамики Wordstat (`GetDynamics`).  
**FR-03.2.** Гранулярность: помесячная частотность.  
**FR-03.3.** Регион: вся Россия (согласно настройкам API-клиента).  
**FR-03.4.** Между запросами — настраиваемая задержка (`request_delay`, по умолчанию 0,4 с).  
**FR-03.5.** При ошибке по отдельному запросу анализ продолжается для остальных.  
**FR-03.6.** Если данных нет совсем — анализ завершается с понятным сообщением об ошибке.

---

### 3.4. Расширение запросов YandexGPT (FR-04, опционально)

**FR-04.1.** При включённом LLM и доступном YandexGPT система предлагает до 8 дополнительных запросов для сравнения.  
**FR-04.2.** Расширенные запросы добавляются к исходному списку (без дубликатов).  
**FR-04.3.** В Excel блок «РАСШИРЕНИЕ ЗАПРОСОВ» содержит **только фразы, по одной на строку**, без маркеров и вступительного текста — для копирования в поле ввода.  
**FR-04.4.** При недоступности GPT расширение пропускается; анализ Wordstat выполняется по исходным запросам.

---

### 3.5. Расчёт метрик (FR-05)

Для каждого запроса система рассчитывает:

| Метрика | Описание |
|---------|----------|
| Скользящее среднее (3 мес.) | Сглаживание частотности |
| Темп роста, % | Помесячное изменение |
| Волатильность | Скользящее стандартное отклонение (12 мес.) |
| Индекс сезонности | Отношение к среднегодовому значению |
| Индекс конкуренции | 0..1, комбинация нормированной частотности и обратной волатильности |
| Импульс роста | Средний рост за последние 6 мес. |
| Средний рост за год | Средний рост за последние 12 мес. |
| Наклон тренда | Линейная регрессия частотности (показов/мес.) |
| Средний рост за весь период | Среднее `growth_pct` за весь интервал |

**FR-05.1.** Волатильность классифицируется: Низкая / Средняя / Высокая (терцили по выборке).  
**FR-05.2.** Запросы без данных Wordstat **всё равно присутствуют** в сводке и таблицах ниш с нулевыми метриками.

---

### 3.6. «Голубые океаны» и «Растущие рынки» (FR-06)

**FR-06.1.** Лист «Голубые океаны» содержит **3 блока** с разными метриками роста:
- импульс (последние 6 мес.);
- средний рост за последние 12 мес.;
- наклон линейного тренда за последние 12 мес.

**FR-06.2.** Лист «Растущие рынки» — аналогично 3 блока.  
**FR-06.3.** В каждом блоке отображаются **все** запросы из списка анализа (включая расширенные GPT).  
**FR-06.4.** Для «Голубых океанов» рассчитывается составной `blue_score` (низкая конкуренция + рост).  
**FR-06.5.** Вердикты по формулам (🟢/🟡/🔴) формируются автоматически.  
**FR-06.6.** При включённом LLM — колонка «Вердикт нейросети (YandexGPT)» для топовых ниш.

---

### 3.7. Прогноз Prophet (FR-07)

**FR-07.1.** Прогноз на **6 месяцев** вперёд по каждому запросу.  
**FR-07.2.** Минимум **12** исторических точек для построения прогноза.  
**FR-07.3.** Результат: `yhat`, нижняя и верхняя границы доверительного интервала.  
**FR-07.4.** При недоступности Prophet или недостатке данных прогноз пропускается без остановки анализа.  
**FR-07.5.** Средний прогнозируемый показ попадает в сводку (колонка «Прогноз сред. показов/мес»).

---

### 3.8. Excel-отчёт (FR-08)

**FR-08.1.** Формат: `.xlsx`, генератор — `xlsxwriter`.  
**FR-08.2.** Оформление **строго по образцу** `yandex_analysis_exemple.xlsx`:
- шрифт Calibri 11;
- синие заголовки `#4472C4`;
- цветные кружки-индикаторы (один символ 🔴 с разным `font_color`: зелёный / жёлтый / красный);
- условная заливка ячеек метрик (конкуренция, волатильность, сезонность).

**FR-08.3.** Состав листов:

| № | Лист | Требования |
|---|------|------------|
| 1 | О файле | Выводы, методология с цветными кружками, блоки GPT (если есть), описание листов |
| 2 | Данные | Помесячные сырые данные + метрики по строкам |
| 3 | Сводка по запросам | Все запросы, колонка № п.п. (как в образце), цветовая индикация конкуренции |
| 4 | Прогноз | Таблица Prophet |
| 5 | График | 2 нативные редактируемые диаграммы Excel |
| 6 | Голубые океаны | 3 блока таблиц |
| 7 | Растущие рынки | 3 блока таблиц |

**FR-08.4.** Имя файла: `yandex_analysis_ГГГГММДД_ЧЧММСС.xlsx`.  
**FR-08.5.** Заголовки таблиц читаются из встроенного образца Excel; при его отсутствии — из встроенных defaults в коде.  
**FR-08.6.** Колонки сводки сопоставляются с заголовками **по смыслу** (не по жёсткому индексу), чтобы корректно работать с колонкой «№ п.п.» из образца.

---

### 3.9. Графики (FR-09)

**FR-09.1.** В Excel — линейные диаграммы: топ-N запросов и все запросы; при наличии прогноза — пунктирные ряды.  
**FR-09.2.** В веб-интерфейсе — интерактивные графики Chart.js с теми же данными (API `/api/chart-data`).  
**FR-09.3.** Параметр `top_n` настраивается (по умолчанию 10).

---

### 3.10. YandexGPT — текстовые блоки (FR-10, опционально)

При включённом LLM и успешном подключении:

| Блок в Excel | Содержание |
|--------------|------------|
| РАСШИРЕНИЕ ЗАПРОСОВ | Дополнительные фразы (построчно) |
| ОБЗОР РЫНКА | Сравнительный обзор всех запросов за период |
| РЕКОМЕНДАЦИИ | Практические рекомендации по топовым «голубым» нишам |
| Вердикт нейросети | По топ-5 запросам на листе «Голубые океаны» |

**FR-10.1.** Пустые блоки не выводятся в Excel.  
**FR-10.2.** Ошибка GPT по одному запросу не прерывает весь анализ.

---

### 3.11. Доставка результатов (FR-11)

**FR-11.1.** Скачивание XLSX через кнопку в интерфейсе (`/api/download`).  
**FR-11.2.** Отправка файла в Telegram (документ), VK (документ), Email (вложение).  
**FR-11.3.** Статус отправки по каждому каналу отображается пользователю.

## 9. Ограничения и допущения

1. Лимиты Wordstat определяются тарифом Yandex Cloud (не более ~100 запросов/час на бесплатном уровне).
2. Период анализа — с 2018 года (ограничение API).
3. Интернет обязателен для сбора данных и GPT.
4. Многопользовательский доступ и разграничение прав не предусмотрены.
5. Образец Excel `yandex_analysis_exemple.xlsx` предоставляется заказчиком и является эталоном оформления.

Продукт считается принятым, если выполнены **все** пункты:

- [ ] С настроенными ключами Wordstat анализ 5+ запросов завершается успешно.

## 12. Глоссарий

| Термин | Определение |
|--------|-------------|
| **Частотность** | Число показов поискового запроса в Яндексе за месяц |
| **Голубой океан** | Ниша с низкой конкуренцией и положительным ростом |
